# Presentations

This directory stores seminar and presentation materials that are separate from
the Sphinx documentation pages.

Files for the pair-execution proposal seminar:

- `generate_gnn_ip_pair_execution_seminar.py`: reproducible PPTX generator
- `gnn_ip_pair_execution_seminar.pptx`: generated slide deck
- `gnn_ip_pair_execution_seminar_script.md`: speaker script

To regenerate the deck:

```bash
.venv/bin/python docs/presentations/generate_gnn_ip_pair_execution_seminar.py
```
